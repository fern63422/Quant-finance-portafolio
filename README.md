# Quantitative Finance Portfolio

This repository contains a selection of academic and personal projects focused on
quantitative finance, investments, and data-driven financial analysis.

The projects apply financial theory, econometrics, and programming (Python) to
real market and macroeconomic data, with an emphasis on portfolio analysis,
risk modeling, and asset pricing.

## Projects Included

- **CAPM and Regression-Based Asset Pricing**
- **Default Risk Modeling**
- ** Macroeconomic Time Series Analysis**

Each project includes:
- A clear problem statement
- Data sources
- Quantitative methods
- Key results
- Financial interpretation

## Tools & Methods
- Python (Pandas, NumPy, Matplotlib)
- Financial econometrics
- Portfolio theory and risk measures
- Regression models (CAPM)
